Engine Architecture

Sentinel AI leverages a state-of-the-art deep learning graph pipeline:

  • Dataset: Juliet Test Suite for robust, varied C/C++ vulnerability paradigms.
  • Vectorization: Word2Vec (asm2vec) instruction-level semantic embedding to capture assembly intent.
  • Graph Modeling: Graph Attention Network (GATv2Conv + global pooling) to inspect CFG connectivity.
  • Evaluation: Stratified 5-Fold Cross-Validation ensuring high-precision vulnerability checks (79.91% AUC-ROC).

How Sentinel Works

01

Binary Upload & Security Validation

Upload a compiled executable or source file. C/C++ source code is automatically compiled on the fly using a MinGW compiler wrapper.

02

Ghidra Extraction

NSA's Ghidra framework performs headless disassembly, traversing basic block nodes and edge connections to extract the function's Control Flow Graph (CFG).

03

Word2Vec Instruction Embeddings

Assembly instruction tokens in each basic block are converted into 128-dimensional dense vectors using a pre-trained asm2vec model.

04

Graph Attention Neural Network

A multi-head GATv2 model processes the graph topology, performing message passing along basic-block edges to compute risk verdicts with extreme accuracy.
