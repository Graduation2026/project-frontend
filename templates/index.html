Engine Architecture

Sentinel AI leverages a robust pipeline tailored for enterprise-grade security:

  • Dataset: Juliet Test Suite for robust, varied C/C++ vulnerability paradigms.
  • Balancing: SMOTE ensures balanced multi-class performance.
  • Vectorization: TF-IDF evaluates opcode structure for implicit flaw detection.
  • Ensemble: Random Forest scales seamlessly for multi-threaded inference.

How Sentinel Works

01

Binary Upload & Security Validation

Upload a compiled executable. The system validates the file signature securely inside an isolated sandbox.

02

Ghidra Extraction

NSA's Ghidra framework performs headless disassembly, traversing functions to extract raw assembly-level opcode DNA.

03

TF-IDF Analytics

Opcode sequences are evaluated using Term Frequency-Inverse Document Frequency to find spatial coding anomalies.

04

AI Classification

A highly-tuned Random Forest ensemble evaluates the vectors against thousands of known CVE patterns.
